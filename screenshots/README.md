# Screenshots

This directory contains screenshots of KeyPilot in action.

To add screenshots:
1. Launch KeyPilot from `/Applications/KeyPilot.app`
2. Take screenshots using `Cmd+Shift+4` or `Cmd+Shift+5`
3. Save them here with these names:
   - `menubar.png` - Menu bar with profile selection
   - `editor.png` - Profile editor window
   - `settings.png` - Settings and configuration view

Recommended dimensions: 800x600 or native resolution
